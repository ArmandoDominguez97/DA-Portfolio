# [Project 1: Global CO<sub>2</sub> Emissions Analysis](https://github.com/ArmandoDominguez97/Global-CO2-Emissions-Analysis)

This project was completed through the COOP Careers Data Analytics Fellowship.

Collaborators: Jiewen Huang, Kevin Elshemy, Hawa Kora

* Data from [World Bank Open Data](https://docs.google.com/spreadsheets/d/1MIM8Gytj1EYGuGaN39PP51qlUBIs7WYp/edit?gid=1354283968#gid=1354283968)
* Data was cleaned by removing null values, standardizing data to begin at the year 1921, and converting CO<sub>2</sub> units from tons to millions of tons for clearer visualizations.
* Tools used: Excel, Tableau, Canva
